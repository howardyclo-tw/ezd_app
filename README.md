# EZD App

A modern web application for managing courses, attendance, and card-based transactions.

## 📖 Documentation

- **[Product Requirements Document (PRD)](docs/prd.md)**: Project overview and core requirements.
- **[Feature Plan & Roadmap](docs/feature-plan.md)**: Detailed implementation phases and progress.
- **[Deployment Guide](DEPLOYMENT.md)**: Instructions for deploying to Vercel.
- **[Development & MCP Tips](docs/mcp-guide.md)**: Guide for developers, including how to use Supabase MCP for agentic coding.

## 🛠 Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Database/Auth**: Supabase
- **Language**: TypeScript

## 🚀 Getting Started

1. Clone the repository.
2. Install dependencies: `pnpm install`.
3. Set up environment variables in `.env.local`.
4. Start the development server: `pnpm dev`.
